# Pixels Bridge for FoundryVTT

Foundry module to handle Pixels devices without passing through WebBluetooth by connecting the dice to an android phone and by forwarding stuff to a relay server. 
I needed this because my adapter was not working fine with Pixels and I'm fundamentally lazy :)

Got most of the code from the official FoundryVTT plugin and adapted it to my necessities: https://github.com/foundryvtt/pixels

**Relay server**: https://github.com/KzMz/pixels-bridge-relay

**React-native app**: https://github.com/KzMz/pixels-bridge-app
